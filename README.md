# Event Rating

Real-time Event Evaluation Platform

## Overview

A web-based application designed to simplify and digitize the evaluation of events using **QR code-based rating collection** combined with **real-time data visualization**. Event organizers can collect instant feedback from participants and monitor results live through dashboards.

This project is developed as part of the **Softwareentwicklungspraktikum (SEP) SS25** at the **Peter L. Reichertz Institute for Medical Informatics**.

---

## Use Cases

Suitable for:

- Academic poster competitions
- Any real-time crowd-based survey or evaluation

### User Roles

- **Participants**

  - Submit ratings anonymously

- ** Organizers**
  - Create and manage events

  - View and analyze results

- Event creation and configuration
- QR code generation per event or evaluation item
- Rating data collection and storage
- Real-time data processing
